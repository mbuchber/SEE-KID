Patient speichern

    

Diese Menüfunktion wird verwendet, um die aktuellen Patientendaten als Datei zu speichern. Navigieren sie in das Verzeichnis, in dem sie ihre Patientendaten ablegen möchten und geben sie der Datei einen entsprechenden Namen. Achten sie darauf, dass sie keinen Namen vergeben, der bereits in dem aktuellen Verzeichnis existiert, um keine Daten zu überschreiben.

 

 

Bestätigen sie den Dialog mit dem Button "Save" bzw. "Speichern", um die aktuellen Daten als Patientendatei abzulegen. Wenn sie den Button "Cancel" bzw. "Abbrechen" betätigen, so wird der aktuelle Patient nicht gespeichert, die Daten vor Aufruf dieser Funktion bleiben aber alle erhalten.
