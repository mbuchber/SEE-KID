Print Preview

    

You can access the print preview of a diagram via its context menu and of the 3D-view via the main menu under "Patient->3D-Model->Print Preview". The print preview is a Windows® default dialog and allows to preview the content to be printed. The specific diagrams are automatically adjusted to the selected paper format.

 

If you want to define a smaller print format or a different print layout, modify the printer's paper size or save the images as files and use e.g. Microsoft Word® or a similar program to design your own printout.
