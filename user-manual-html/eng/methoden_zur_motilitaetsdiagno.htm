Methods for Motility Diagnosis

    

In SEE++ there exist two possibilities for the interactive motility diagnosis:

 

1.Interpolation for Real-Time Assessment

2.Cover Test

 

You can access these two functions via the "General Functions" toolbar. As long as one of these two functions is activated, you cannot make changes in SEE++ which would lead to a recalculation of the simulation results (e.g. a change of the medical data or the loading of another scenario). The motor fusional ability forms an exception as it can still be modified when one of those functions is activated, since it is used for the simulation of the cover test.
