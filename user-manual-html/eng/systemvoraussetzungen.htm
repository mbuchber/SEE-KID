System Requirements

    

SEE++ runs on computers with a Windows® operating system installed.

Due to the graphically extensive representation and the computationally complex mathematical procedures used by SEE++, certain minimum requirements apply to a computer system running the software. These requirements should be fulfilled to guarantee a trouble-free operation of the software.

 

Requirements for SEE++:

 

•Operating system: Windows® 2000 or higher

•Intel Pentium 4 with 2 GHz minimum (or Intel Core architecture with 1.4 GHz minimum)

•512 MB RAM minimum

•nVidia GeForce 4 with 64 MB RAM minimum or ATI Radeon 9000 with 64 MB RAM minimum

•Monitor or TFT display with a resolution of 1024x768 pixel minimum with True Color
